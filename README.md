# 家計簿アプリ (Kakeibo Expense Tracker)

日本の伝統的な家計簿の考え方に基づいた、シンプルで美しい支出管理アプリです。

## 機能

- 📝 **支出記録** - 日付、金額、店名、カテゴリ、支払い方法をワンタップで記録
- 📋 **履歴管理** - 日ごとにまとまった支出履歴を確認・編集・削除可能
- 📊 **統計分析** - カテゴリ別・支払い方法別の支出集計とグラフ表示
- 💾 **ローカル保存** - ブラウザーLocalStorageに自動保存、データはお使いのデバイスに保管
- 📱 **レスポンシブ** - モバイル、タブレット、PCに対応
- 🌸 **和風デザイン** - 和色を使用した落ち着きのある日本的なUI

## 使い方

1. **記録タブ** - 支出を新規記録
2. **履歴タブ** - 過去の記録を確認・編集・削除
3. **集計タブ** - 月別の支出分析を表示

## 技術仕様

- HTML5 + CSS3 + Vanilla JavaScript
- LocalStorageを使用したクライアントサイド保存
- 外部依存ライブラリなし（スタンドアロン）
- 日本語対応
